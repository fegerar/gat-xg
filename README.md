# gat-xg

Graph Attention Networks for Interpretable Expected Goals Analysis in Soccer

### Report

Read the pdf report on this [link](https://github.com/fegerar/gat-xg/blob/main/report.pdf)

### Dataset
The dataset is built from the Statsbomb dataset. The idea is to get all the ball possessions from all the teams from all the games in the dataset.

### Alternative Attention

I propose an alternative attention formula to improve the interpretability of the model
